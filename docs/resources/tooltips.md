# Contenu pour Infobulles (Tooltips) - Disjoncteur Intelligent (WRS)

Ce fichier contient le texte exact à intégrer dans les infobulles de l'interface utilisateur lorsque le protocole "Risque Weekend" est actif.

---

**ID de l'Infobulle : `WRS_ACTIVE`**

**Texte :**
"MODE DE PROTECTION WEEKEND ACTIF : Les nouveaux trades LONG sont temporairement suspendus en raison d'un risque élevé de volatilité détecté sur le marché."
