## Pour Commencer

Bienvenue sur HarmoFinder. Voici le processus simple pour configurer votre compte et être prêt à recevoir des signaux de trading.

1.  **Connectez-vous :** Accédez à la plateforme en utilisant vos identifiants. Nous utilisons une méthode de connexion sécurisée pour protéger votre compte.
2.  **Connectez Votre Échange :** Allez à la page "Compte" ou "Paramètres". Vous devrez générer des clés API depuis votre échange de cryptomonnaies préféré (par exemple, Bitget, Binance). Suivez les instructions à l'écran pour ajouter vos clés API. *(Note : Pour des raisons de sécurité, assurez-vous que les autorisations de l'API sont réglées sur "Trader" et "Lire" uniquement. N'activez jamais les autorisations de "Retrait".)*
3.  **Configurez Votre Risque :** Dans votre tableau de bord, définissez vos paramètres de risque souhaités. Cela inclut généralement le "Risque par Trade" (par exemple, 1% de votre compte) et l'"Effet de Levier Maximal". Le bot utilisera ces paramètres pour calculer automatiquement la taille des positions.
4.  **Activez le Trading :** Une fois configuré, vous pouvez activer le bot depuis votre tableau de bord principal. Le bot commencera alors à surveiller les nouveaux signaux de trading à haute confiance pour les exécuter en votre nom.
