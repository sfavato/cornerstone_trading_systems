# Analyse Fondamentale Intégrée

Pour les crypto-actifs, l'analyse technique seule est insuffisante. Nous avons intégré des métriques fondamentales et on-chain directement dans l'interface pour fournir une vue à 360 degrés.

## Métriques Clés Disponibles

Chaque page d'actif inclut un onglet "Analyse Fondamentale" avec les données suivantes :

- **Données On-Chain :**
    - **MVRV (Market Value to Realized Value) :** Pour identifier si l'actif est surévalué ou sous-évalué.
    - **Net Unrealized Profit/Loss (NUPL) :** Mesure le sentiment général du marché (Euphorie, Anxiété, Capitulation).
    - **Stock-to-Flow (S2F) Model :** Pour le Bitcoin, compare la rareté à la production.

- **Données Financières (pour les actions tokenisées) :**
    - **Ratios P/E (Price-to-Earnings) et P/S (Price-to-Sales) :** Pour une évaluation de base.
    - **Données de Revenus Trimestriels :** Visualisées pour identifier la croissance.

## Cas d'Usage Stratégique

- **Filtre Macro :** Utilisez ces données pour éviter de trader à contre-courant d'une tendance fondamentale. Un signal d'achat technique sur un actif avec un MVRV indiquant une surévaluation extrême est un trade à faible probabilité.
- **Allocation de Taille de Position :** Augmentez la taille de votre position lorsque l'analyse technique et fondamentale sont alignées.
